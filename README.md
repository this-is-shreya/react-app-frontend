# Borrow-Request-App

## This application has three screens:
1. Sign Up
2. Login
3. Borrow request screen

## Technologies/frameworks used:
1. Node.js
2. Express.js
3. React.js
4. MongoDB
5. Fast2SMS for OTP verification
